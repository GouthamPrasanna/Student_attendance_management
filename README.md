# Student Attendance Management System

A full-stack **Student Attendance Management System** built with **Python**, **JavaScript**, **MongoDB**, and **PyMongo** to simplify attendance tracking and student record management.

## Features

* 📌 Student registration and management
* ✅ Attendance marking and tracking
* 🔄 Full CRUD operations for student and attendance records
* 📊 Attendance history and report generation
* 📁 Export attendance reports to CSV
* 💾 MongoDB integration using PyMongo for efficient data storage
* 🎨 Responsive JavaScript-based user interface

## Tech Stack

* **Backend:** Python
* **Frontend:** HTML, CSS, JavaScript
* **Database:** MongoDB
* **Database Driver:** PyMongo

## Key Highlights

* Designed a modular full-stack architecture with a Python backend and JavaScript frontend.
* Implemented efficient MongoDB queries for fast data retrieval and updates.
* Automated report generation to reduce manual effort and improve record management.
* Built a user-friendly interface for seamless attendance management.

This project demonstrates full-stack development, database integration, CRUD operations, and practical application development using Python and MongoDB.
